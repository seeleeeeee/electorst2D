#  EStatix — 2D Electrostatic Field Simulation

**EStatix** — приложение для расчета и визуализации электростатических полей методом конечных элементов (FEM).

**⚠️ Текущая версия: v1.2 (Preview)** — базовая функциональность готова, настоящий FEM-решатель в разработке.

##  Управление

| Действие | Горячая клавиша |
|----------|-----------------|
| Точка | `N` |
| Линия | `L` |
| Выделение | `S` |
| Построить сетку | `Ctrl+M` |
| Анализ (линейный потенциал) | `Ctrl+R` |
| Сохранить геометрию | `Ctrl+S` |
| Загрузить геометрию | `Ctrl+O` |
| Сохранить всё (сетка + результаты) | `Ctrl+Shift+S` |
| Загрузить всё | `Ctrl+Shift+O` |
| Очистить всё | `Ctrl+C` |
| Приблизить/Отдалить | `Колесо мыши` / `Ctrl++` / `Ctrl+-` |

##  Как использовать (текущая версия)

1. Нажмите **N** и создайте 3 точки
2. Нажмите **L** и соедините точки по кругу (1→2, 2→3, 3→1)
3. Нажмите **Ctrl+M** — построится сетка внутри фигуры
4. Нажмите **Ctrl+R** — выполнится расчет **линейного потенциала** (заглушка, не FEM)
5. Настройте граничные условия в левой панели

##  Сборка

### Требования
- Qt 5.15.2 (MinGW 32-bit)
- MinGW 8.1.0

### Команды
```bash
cd EStatix
qmake electrostatic2d.pro
mingw32-make -j4
cd release
./electrostatic2d.exe
