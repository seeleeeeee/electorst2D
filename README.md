# EStatix — 2D Electrostatic Field Simulation

**EStatix** — приложение для расчета и визуализации электростатических полей методом конечных элементов (FEM).

**Версия: v1.3** — реальный FEM-решатель, материалы, цветная визуализация.

##  Как работает FEM-решатель

Программа решает **уравнение Пуассона**:
∇·(ε ∇φ) = -ρ

**Метод решения:**
1. Генерация треугольной сетки
2. Сборка локальных матриц жесткости
3. Учет диэлектрической проницаемости: `factor = ε·ε₀ / (4·Area)`
4. Сборка глобальной разреженной матрицы
5. Учет граничных условий методом штрафа
6. Решение СЛАУ (Eigen::SimplicialLDLT)
7. Визуализация потенциала

##  Как использовать

1. Нажмите **N** и создайте 3-4 точки
2. Нажмите **L** и соедините точки по кругу
3. Нажмите **Ctrl+M** — построится сетка
4. Выберите материал в левой панели
5. Настройте граничные условия
6. Нажмите **Ctrl+R** — решатель выполнит расчет
7. Результат отобразится цветной картой
